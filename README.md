## 🚀 快速体验

### 🔧 本地部署
```bash
# 获取源码
git clone <your-repo> html-paste-host
cd html-paste-host

# 安装依赖
npm install

# 启动服务 🎉
npm start

# 访问应用
open http://localhost:3000
```

### 🐳 Docker 部署（推荐）
```bash
# 一键启动 Premium 版本
docker compose up -d --build

# 或使用单独容器
docker build -t html-paste-host:premium .
docker run -d -p 80:3000 \
  -v $(pwd)/pages:/app/pages \
  --name html-paste-host-premium \
  html-paste-host:premium
```

### ⚡ 快速更新
```bash
# 自动更新脚本
./update.sh

# 健康检查
curl http://localhost:3000/api/health
```

## 📚 使用指南

### 🎨 **创作工作流**

#### 1️⃣ 创建页面
- 点击 **"创建新页面"** 或使用 `Ctrl+N`
- 在双面板编辑器中编写 HTML
- 选择缩略图：随机精选 | 网络链接 | 本地上传
- 实时预览，`Ctrl+S` 保存

#### 2️⃣ 管理内容
- **标签分类**: 点击标签图标添加分类标签
- **快速搜索**: `Ctrl+F` 聚焦搜索框
- **筛选排序**: 按时间、热度、名称等维度排序
- **状态追踪**: 查看访问量、更新状态

#### 3️⃣ 分享作品
- 悬停页面卡片，点击 **分享按钮**
- 复制链接、生成二维码、获取嵌入代码
- 支持社交媒体分享

### ⌨️ **快捷键大全**
```
全局快捷键:
Ctrl+N     → 新建页面
Ctrl+F     → 聚焦搜索
Esc        → 关闭模态框

编辑器快捷键:
Ctrl+S     → 保存页面  
Ctrl+Enter → 刷新预览
Tab        → 增加缩进
Shift+Tab  → 减少缩进
```

### 🎯 **高级技巧**
- **批量管理**: 使用标签系统对页面进行分类管理
- **数据导出**: 定期导出数据备份，支持 JSON 格式
- **性能优化**: 利用访问统计分析热门内容
- **主题切换**: 根据时间或个人喜好切换明暗主题

- **⚡ 交互体验**: 防抖搜索、快捷键、通知系统

- **🚀 Runtime**: Node.js + Express (ES Modules)
